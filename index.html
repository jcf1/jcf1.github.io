LexiLadder (2024)

A Wordle-inspired daily web game that challenges one's vocabulary. Connect the start word to the end word by building a chain (or ladder) of synonyms, chaining together common synonyms until you reach the goal or run out of guesses. Give it a try and put your vocabulary to the test! Frontend was written in Svelte (Diagrams were made using SvelteFlow library) and backend is a Go server that connects to a PostgresSQL database, Redis cache, and external dictionary API.

Loop-Finder (2023)

A program that attempts to find repeating or "looping" gifs from a video using locality-sensitive hashing. I created a web app that allows a user to upload a video and select a portion of the video to search for loops. I also wrote a command-line program with additional parameters and the ability to run on either a portion of or the whole video. Image hashing code I used is https://github.com/JohannesBuchner/imagehash.

Sharewatch: Communal Stopwatch (2022)

Sharewatch is an app avilable on the Apple App Store that allows a user to start multiple stopwatches for multiple users at the same time and have them all synced up. One user can create a room that others can join using the generated room code; the first user is then able to start all the connected watches at the touch of a button.

Infinite Block Puzzle (2020)

A simple tile-based puzzle web game where the user navigates a block to the goal by sliding and colliding with blocks. All the puzzles are procedurally generated using a seed the can be input by the user and the puzzle can be made more difficult by an option to add additional blocks that aren't used in the solution.
